Before you read
Are you in the right place?
This book describes a specific pattern of chronic multi-system illness. Before reading, take a few minutes with one of these validated clinical questionnaires. They will help you and your doctor understand whether what you are experiencing matches this pattern.
Important — please read before you start
This tool does not diagnose. Depression, anxiety, thyroid problems, anaemia, and many other treatable conditions can produce similar symptoms. If you have not yet seen a doctor, please do that first. This assessment is for people who have already done that and are still without answers.
A note on honesty
These questions are only useful if you answer them honestly. It is easy to read “do you sweat more than normal” and say yes because it has been a warm summer. Before answering each question ask yourself — is this genuinely different from how I was before? Is it happening in contexts where it shouldn't? Has it been consistent for months? If the honest answer is no — answer no. These tools help you if you use them honestly. They mislead you if you don't.
